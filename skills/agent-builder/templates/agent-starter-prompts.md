# Agent starter prompts

Starter prompts should demonstrate real tasks, not generic greetings.

## Prompts

1. `<prompt for the most common workflow>`
2. `<prompt for a more advanced workflow>`
3. `<prompt that exercises knowledge sources>`
4. `<prompt that exercises tools/actions, if available>`
5. `<prompt that tests boundaries or escalation>`

## Quality rules

- Use language a real user would type.
- Include enough context to show the expected workflow.
- Avoid prompts that require private data unless they are examples with placeholders.
