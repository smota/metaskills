# Examples

These examples show how `metaskills` turns broad goals into reusable, reviewable agent and skill assets.

| Example | Use this when | Skill |
| --- | --- | --- |
| [Build release notes agent](./build-release-notes-agent/) | You want an agent that drafts release notes from issues and changelog fragments | `agent-builder` |
| [Create support triage skill](./create-support-triage-skill/) | You want to convert a support workflow into an installable skill | `skill-creator` |
| [Database migration review agent](./database-migration-review-agent/) | You want an agent that reviews risky database changes before release | `agent-builder` |
| [Repository onboarding agent](./repository-onboarding-agent/) | You want a knowledge-only agent for new contributors | `agent-builder` |

Each example includes a starting prompt, expected output structure, value, assumptions, and a validation checklist.
