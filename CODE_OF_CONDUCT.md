# Code of Conduct

This project follows a simple standard: be respectful, constructive, and focused on helping people build better reusable agent capabilities.

## Expected behavior

- Use welcoming and inclusive language.
- Respect different tools, workflows, and levels of experience.
- Keep feedback specific, actionable, and kind.
- Prefer evidence, examples, and validation over personal preference.
- Do not share secrets, private data, or proprietary customer information.

## Unacceptable behavior

- Harassment, threats, or discriminatory language.
- Personal attacks or sustained disruption.
- Publishing private information without permission.
- Introducing malicious instructions, hidden exfiltration, or unsafe agent behavior.

## Reporting

Open a GitHub issue or contact the maintainer through the repository owner profile if you need to report a concern. Maintainers may edit, remove, or reject contributions that violate this code.
