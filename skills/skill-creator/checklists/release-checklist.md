# Release checklist

- [ ] Skill package is self-contained.
- [ ] Required templates live inside the skill directory.
- [ ] References are linked with relative paths.
- [ ] Evals or acceptance criteria are included.
- [ ] Benchmark decision is recorded for material behavior changes.
- [ ] Learning log is updated if enabled.
- [ ] Telemetry plan is absent or explicitly opt-in.
- [ ] No secrets, raw private data, or customer data are committed.
- [ ] Installation path is documented.
