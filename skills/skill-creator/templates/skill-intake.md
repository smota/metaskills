# Skill intake

## Goal

What should this skill help an agent do?

## Target users

Who will use the skill?

## Trigger phrases

What should activate the skill?

## Workflow

What steps should the agent follow?

## Inputs

What information, files, tools, or context does the skill need?

## Outputs

What should the skill produce?

## Boundaries

What should the skill refuse, avoid, or escalate?

## Examples

Provide realistic prompts and expected outcomes.

## Validation

How will we know the skill works?
